# Online Scam Pattern Detector

A machine learning based system that detects potential online scam patterns from emails, images, and browser activity.

## Features
- Detect scam patterns using ML
- Email scam detection
- Image scam detection
- Browser extension monitoring
- Real-time alert system

## Tech Stack
- Python
- Flask
- Machine Learning
- HTML / CSS / JavaScript

## Project Structure
app.py – main Flask server  
database.py – database operations  
email_service.py – email scanning module  
image_scanner.py – image scam detection  
news_feed.py – scam news tracking  

Folders:
templates/ – HTML files  
static/ – CSS, JS  
ml/ – machine learning models  
patterns/ – scam pattern data
